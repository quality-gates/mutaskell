data MuOp Source #

A mutation operation: a before/after pair of the same node type. Guard mutations use A (the parent Alt_) because GRHS lacks an Outputable instance, making the whole-match replacement the simplest strategy that works with the MuNode constraint.

(~~>) :: (MonadPlus m, MuNode a) => a -> a -> a -> m a Source #

Replace a specific node occurrence.

Matching is by source location (getHasLoc) rather than by value equality, because GHC's AST types do not derive Eq. Once a match is found, transferEntry copies the original node's leading-whitespace delta onto the replacement so that exactPrint preserves layout.

same :: MuOp -> Bool Source #

same op is True when the before and after nodes serialise to the same string — i.e., the mutation is a no-op and should be discarded. Uses ppr so no Eq instance on AST nodes is required.

type MuNode a = (Typeable a, HasLoc a, CanTransfer a, Outputable a) Source #

Constraint alias for everything a node type needs to participate in mutation: generics traversal (Typeable), source-location access (HasLoc), annotation repair (CanTransfer), and pretty-printing (Outputable).

class CanTransfer a where Source #

Typeclass that captures the ability to transfer an entry delta from one located node to another. All MuOp node types are GenLocated (EpAnn t) a for some Typeable t, which makes the transfer possible via transferEntryDP.
