relevantOps :: Data a => a -> [(MuVar, MuOp)] -> [(MuVar, MuOp)] Source #

Filter out identity operations (where before === after by ppr). The applicability check is deferred to once in mutate, which returns an empty list for operators that do not match any AST node.

once :: forall (m :: Type -> Type). MonadPlus m => GenericM m -> GenericM m Source #

Apply a mutating function on a piece of code exactly once at the first matching site found in a depth-first traversal. Skips sub-trees rooted at structural or declarative nodes (see isSkippedDecl) to avoid generating non-compilable mutants.
